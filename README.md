## Information

- [x] Cache by phpfastcache/phpfastcache ^8.0
- [x] Debug by nguyenanhung/my-debug ^2.0
- [x] Support Driver:`files`, `apcu`, `redis`, `predis`, `memcache`, `memcached`,`mongodb`,`cassandra`, `couchbase`, `couchdb`
